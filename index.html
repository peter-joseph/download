I created this template to use with my own projects that I might be offering for download. No strings attached, you can do what you like with it.

How do I use this Template?

If you know HTML and CSS then you should have no problems with making changes to the template itself, However, It is simple enough that using a WYSIWYG editor such as Dreamweaver should be sufficient.

The main components that you will want to change when editing the template:

  • The meta tags in <head>
  • The Download title in #Header
  • The Subtitle contained in #about
  • The Download links to your file #Download
  • The extended description of the file contained in the <p> tags.

Other things you may want to know...

  • The styling of the buttons 'Download' and 'Demo' apply directly to links contained inside the div #download. If you create a link inside this div, it will automatically be styled as a button similar in appearance to what you above. If you would like to use the button style outside this div, you can use the following css class .button
  • You can add code snippets into your template. To do so, wrap the code as follows. 
    <pre><code>Code goes in here...</code></pre>
  • The width of the div #container controls the width of the entire document.
